# Dental Clinic System

A modern clinic management platform for dental practices — appointments, patients, treatments, billing, and operational dashboards in one place.

## Why this exists

Dental clinics lose time through scattered appointment books, manual patient tracking, and unclear billing workflows. This system provides a clean, centralized workflow for front-desk teams, dentists, and clinic managers.

## Key Features

- Patient management
- Appointment scheduling
- Dentist and staff management
- Treatment records
- Invoices and payments
- Clinic dashboard
- Role-based access
- Search and filtering
- Responsive UI
